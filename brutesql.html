Inserting Data

When inserting data we set insert property to target table, all other properties will be inserted to this table, and if properties do not match with existing columns, new column is created and old rows are updated with value NULL.
let data = { insert: users, name: name, email: email }

Create Table

We can create table ourselves by sending object containing property create.
let data = { create: users };
Other option is just to use property insert which forces creation of table automatically if table does not exist.
let data = { insert: users };
